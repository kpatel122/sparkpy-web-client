Create Trails

To start using trails create a Trails() Object

import sparkpy

#create a trails object
t = sparkpy.Trails()

Movement

To move a trails character, use Forward() and Backward()

import sparkpy

#create a trails object
t = sparkpy.Trails()

#move the trails object forward by a distance of 1 unit
t.Forward(1)

Rotation

To rotate a trails character, use Right() and Left()

import sparkpy

#create a trails character
t = sparkpy.Trails()

#move the trails character forward by a distance of 1 unit
t.Forward(1)

#turn left by 90 degrees
t.Left(90)

#move the trails character forward by a distance of 1 unit
t.Forward(1)

Pen Up & Pen Down

To stop leaving trails as the character moves use PenUp(). To resume trails use PenDown()

import sparkpy

#draw 2 parallel lines

#create a trails character
t = sparkpy.Trails()

#move the trails character forward by a distance of 1 unit
t.Forward(1)

#turn left by 90 degrees
t.Left(90)

#stop trails effect
t.PenUp()

#move the trails character forward by a distance of 0.5 unit
t.Forward(0.5)

#turn left
t.Left(90)

#resume trails effect
t.PenDown()

#move the trails character forward by a distance of 1 unit
t.Forward(1)

Trail Colour

To set the colour of the trail, use PenColour

import sparkpy

#create a trails character
t = sparkpy.Trails()

#set the trail colour to blue
t.PenColour("blue")

#move the trails character forward by a distance of 1 unit
t.Forward(1)

#turn left by 90 degrees
t.Left(90)

#move the trails character forward by a distance of 1 unit
t.Forward(1)

Note

A full list of colours can be found at Creating effects

Trail Size

The size of the trail can be set using PenSize()

import sparkpy

#create a trails character
t = sparkpy.Trails()

#move the trails character forward by a distance of 1 unit
t.Forward(1)

#turn left by 90 degrees
t.Left(90)

#change the colour to blue
t.PenColour("blue")

#make the trail half the size
t.PenSize(0.5)

#move the trails character forward by a distance of 1 unit
t.Forward(1)
