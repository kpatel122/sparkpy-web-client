Creating effects

The Effect() object will create a special effect, it’s default position is 0 The colour of the effect can be set with SetColour() An effect can be stopped with Stop()

import sparkpy

#special effects example

#create office environment
sparkpy.Environment("Office")

x = 0 #x position
y = -1 #y position
z= 0 #z position

#create a portal effect
portal=sparkpy.Effect("portal",x,y,z)

#set colour to red from default
portal.SetColour("red")

Note

The current list of effects:

"portal"

The current list of colours:

"yellow"

"clear"

"grey"

"magenta"
