Create an environment
"""""""""""""""""""""
To create a scene an `Environment() <Source/sparkpy.html#sparkpy.Environment>`_ object is needed ::
   
   import sparkpy

   #create a desert enviroment
   sparkpy.Environment('desert')

   A list of all current environments
